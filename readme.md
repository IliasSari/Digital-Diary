A simple yet powerful **C#** application for the Linux Terminal, combining a personal reflection diary with an English practice Quiz.

# ALWAYS RUN IT ON TERMINAL WITH dotnet run COMMAND!
## 🚀 Features
* **Personal Diary:** Log your thoughts with automatic date and time stamps.
* **Persistent Storage:** Entries are saved permanently to a `.txt` file (`diary.txt`).
* **Interactive Menu:** Smooth navigation using a robust `switch` statement logic.
* **History Viewer:** Read all your past reflections directly within the console.

## 🛠️ Technologies
* **Language:** C# 12.0 / 13.0
* **Framework:** .NET 8.0 / 9.0 (Console Application)
* **OS:** Linux (Ubuntu/Debian/Arch)
* **Editor:** VS Code
